Local platform

A daemon for ingestion, a CLI for inspection, and MCP for agents.

Run codepulse doctor, codepulse status, codepulse registry, or codepulse daemon. The local daemon listens on HTTP 127.0.0.1:7842, and the MCP server lets any AI agent query your stats.

terminal 
$ codepulse doctor
✓ data-directory        ~/.codepulse
✓ auth-token            present, mode 0600
✓ database              SQLite ready
✓ daemon-reachable      http://127.0.0.1:7842
✓ registry              6 scanners

Privacy / local-first

Your data never leaves your machine.

Code Pulse has no cloud account, no telemetry pipeline, and no remote analytics service. Activity lands in a local SQLite database and stays there.
