# CSV/Excel Export Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let users export sessions as CSV (RFC 4180) in addition to the existing JSON export. CSV opens directly in Excel/Numbers/Google Sheets — this unlocks invoicing, timesheets, and ad-hoc analysis.

**Architecture:** A pure `csvExporter.ts` module converts an array of session rows into a CSV string. The existing `codepulse.exportData` command gains a format picker; a new `codepulse.exportDataAs` command accepts format as an argument for scripting.

**Tech Stack:** TypeScript only. No new deps. CSV escaping per RFC 4180 (quote fields containing comma, quote, or newline; double internal quotes).

- [ ] **Step 3: Implement csvExporter.ts**

```typescript
import { DatabaseSession } from '../DatabaseManager';

const COLS = [
    'id', 'start', 'end', 'duration_min', 'idle_min',
    'project', 'language', 'file', 'branch',
    'keystrokes', 'lines_added', 'lines_removed', 'productivity'
];

export function escapeCsvField(value: string | number | undefined | null): string {
    if (value === undefined || value === null) return '';
    const s = String(value);
    if (s === '') return '';
    const needsQuote = /[",\n\r]/.test(s);
    if (!needsQuote) return s;
    return `"${s.replace(/"/g, '""')}"`;
}

export function sessionsToCsv(sessions: DatabaseSession[]): string {
    const header = COLS.join(',');
    const rows = sessions.map(s => {
        const durMin = Math.round((s.duration || 0) / 60000);
        const idleMin = Math.round((s.idleDuration || 0) / 60000);
        const fields = [
            s.id,
            s.startTime.toISOString(),
            s.endTime ? s.endTime.toISOString() : '',
            durMin,
            idleMin,
            s.project,
            s.language,
            s.file,
            s.branch ?? '',
            s.keystrokes ?? 0,
            s.linesAdded ?? 0,
            s.linesRemoved ?? 0,
            s.productivityScore ?? ''
        ];
        return fields.map(escapeCsvField).join(',');
    });
    return [header, ...rows].join('\n');
}
```

```typescript
public async exportData(format: 'json' | 'csv' = 'json'): Promise<void> {
    try {
        const ext = format === 'csv' ? 'csv' : 'json';
        const defaultName = `codepulse-export-${formatLocalDate(new Date())}.${ext}`;
        const filters = format === 'csv'
            ? { CSV: ['csv'] }
            : { JSON: ['json'] };

        const uri = await vscode.window.showSaveDialog({
            defaultUri: vscode.Uri.file(defaultName),
            filters
        });
        if (!uri) return;

        let content: string;
        if (format === 'csv') {
            const { sessionsToCsv } = await import('../storage/exporters/csvExporter');
            const endDate = new Date();
            const startDate = new Date(); startDate.setFullYear(2000);
            const sessions = await this.databaseManager.getSessionsByDateRange(startDate, endDate);
            content = sessionsToCsv(sessions);
        } else {
            const data = await this.databaseManager.exportAllData();
            content = JSON.stringify(data, null, 2);
        }

        await vscode.workspace.fs.writeFile(uri, Buffer.from(content));

        if (this.configManager.shouldShowNotifications()) {
            void vscode.window.showInformationMessage(`Data exported to ${uri.fsPath}`);
        }
    } catch (error) {
        const logError = error instanceof Error ? error : new Error(String(error));
        this.logger.error('Failed to export data', logError);
        vscode.window.showErrorMessage('Failed to export data.');
    }
}
```

## Out of scope

- XLSX binary format (JSON/CSV cover the 95% case; XLSX requires a 200KB+ dep).
- Activity/segment/rollup CSV export (sessions are the only row-oriented data users typically want in a spreadsheet).
- Date-range selector in the export flow (future).
