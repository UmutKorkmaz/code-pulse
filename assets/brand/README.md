# Code Pulse Brand Assets
Palette: teal `#34e0a1` to `#22c386`, coral `#f07a5f`, tile `#0e1116` to `#151c25`.
Ink on dark is `#f4f7f6`; muted support text is `#9aa7b2`.
Use `codepulse-tile.svg` for app icons, VS Code icon, desktop icon, and favicon source.
Use `codepulse-mark.svg` for compact inline marks on dark or transparent UI.
Use `codepulse-lockup.svg` when the brand name must travel with the mark.
Clear space: keep at least one pulse stroke width around the mark or tile.
For the lockup, keep clear space equal to the height of the C counter area.
Minimum sizes: tile 16px, mark 16px, lockup 140px wide.
Prefer dark backgrounds; the mark is tuned for `#0e1116` to `#151c25` surfaces.
Do not recolor, stretch, rotate, outline, or redraw the pulse.
Do not add shadows, blur, extra glow, badges, or effects beyond the source SVG.
Do not replace the wordmark font stack unless converting it to final vector paths.
